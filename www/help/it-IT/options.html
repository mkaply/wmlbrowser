Wmlbrowser opzioni

In questa sezione:
  • Indicare al sito web che il browser intepreta i contenuti WML
  • Caricamento dei file WML dal filesystem

Queste sono le opzioni di wmlbrowser che potete regolare nella versione 0.7.

Indicare al sito web che il browser interpreta i contenuti WML

Cosa è questa opzione?

Ogni volta che trasmettete una richiesta per una pagina nel web, il vostro browser trasmette qualcosa denominato Intestazione di accettazione, che genera il contenuto in grado di essere interpretato dal browser.

Se spuntate questa opzione, il vostro browser trasmetterà che è in grado di interpretare contenuti WML.

Ciò può indurre il web server a trasmettergli una pagina differente.

Quale è l'impostazione consigliata?

L'impostazione migliore per questo, di default, è disinserito. Spuntandola, potreste avere pagine web non previste e riportate come errore su Hotmail.

Gli utenti di Firefox possono anche cambiare questa opzione aggiungendo "un tasto di uso di WML" alla toolbar. (Tasto destro sulla toolbar e scegliere "Personalizza", quindi trascinare il pulsante di WML (icona del telefono mobile) sulla toolbar.)

Gli utenti di SeaMonkey possono abilitare il pulsante nella sezione "wmlbrowser" della finestra Preferenze.

Caricare files dal filesystem

Quale è il problema?

Di default, l'estensione wmlbrowser permette Mozilla/Firefox di caricare e visualizza pagine WML da un web server. Ma se avete un file sul vostro disco con una estensione ".wml", il vostro browser non sa cosa fare con esso.

Se selezionate questa opzione, il vostro browser saprà che un file .wml contiene contenuto WML e lo visualizzerà correttamente.

Quale è l'impostazione consigliata?

Io suggerisco di impostare a off questa opzione a meno che non sappiate caricare file WML dal vostro filesystem.
