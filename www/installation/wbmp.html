WBMP support in Firefox

Install wbmpviewer 0.2.7
for Firefox 3.0 on Windows/Linux

Install wbmpviewer 1.1.0
for Firefox 3.5 on Windows/Linux


WBMP is an image format used on wireless devices alongside WML.

Currently there are XPIs available for Firefox 3 or 3.5 on Windows and Linux only. I do not have the resources to create a Mac version.
