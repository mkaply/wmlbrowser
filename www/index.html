wmlbrowser 0.7.26 is released - install it here

wmlbrowser 0.7.26 has been released. This is compatible with Firefox 4. It is compatible back to Firefox 3.

Users who installed from this site can upgrade automatically to the new version.

wbmpviewer 1.1.0 is released - install it here

wbmpviewer 1.1.0 has been released for Firefox 3.5 on Windows/Linux. Despite the increase in version number, there are no functional changes in this release.

Due to changes in Firefox, there will be no further releases of wbmpviewer.

This project is intended to try and implement as much of WML (Wireless Markup Language) as possible in Mozilla/Firefox. The current extension is based on Raoul's attachments to Mozilla bug 35995.

Associated extensions

You may also find the following extensions useful when using wmlbrowser:

  • User Agent Switcher
  • Small Screen Renderer

Help wanted

If you want to help, then feel free to have a look at the bug list. Or, if you're in a position to compile extensions, then that's needed for WBMP support. Or, if you're not much of a coder, then you could always contribute some test cases. Finally, if you're a graphics wizard, you could consider contributing an icon for wmlbrowser.
